Waste Tracking GUI

This application is my first foray into using TKinter with python. It is a simple program that takes user inputs and puts them into a Google Sheet. I was driven to create this project because one of my classes involved tracking the amount of paper waste I generated. This program made it super easy to just run the code and add entries to the sheet without having to open up my web browser.
